# Python Course Lab Solutions

This repository contains my solutions for the Python programming course labs at Polytech Nice-Sophia. These solutions are implemented in **pure Python** (using only the Standard Library) to demonstrate foundational algorithmic thinking and object-oriented programming.

| Lab | Primary Topic | Featured Concepts & Exercises |
| --- | --- | --- |
| **Lab 1** | **Foundations** | Unit conversion logic, quiz grading from CSVs, and recursive decision trees (animal guessing game). |
| **Lab 2** | **Graph Theory** | Graph representation using adjacency lists, connectivity checks, cycle detection, and Breadth-First Search (BFS). |
| **Lab 3** | **OOP & Classes** | Inheritance (Dates), custom properties (Oil Wells), and pathfinding robots using coordinate-based movement. |
| **Lab 4** | **Data Structures** | Custom Stack implementation, bracket balance checking, and iterative Depth-First Search (DFS). |
